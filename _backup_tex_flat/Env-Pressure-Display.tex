\begin{abstract}
We propose a display-only rescale for environment-dependent observables,
$E' = E\,\varphi^{\beta P}$, that leaves the integer scaffolding intact. The
coefficient $\beta$ is fixed by eight-window neutrality of the observable and is
not a tunable parameter. No changes to the core RS integers are permitted.
\end{abstract}

\section{Definition}
Given a base observable $E$, environment coordinate $P$, and fixed $\beta$,
define $E' = E\,\varphi^{\beta P}$. Neutrality is tested by aligned sums over
windows of eight steps in the measured stream.
